#!/bin/sh #

  1. An example hook script to check the commit log message.
  2. Called by “git commit” with one argument, the name of the file
  3. that has the commit message. The hook should exit with non-zero
  4. status after issuing an appropriate message if it wants to stop the
  5. commit. The hook is allowed to edit the commit message file.
#
  1. To enable this hook, rename this file to “commit-msg”.

  1. Uncomment the below to add a Signed-off-by line to the message.
  2. Doing this in a hook is a bad idea in general, but the prepare-commit-msg
  3. hook is more suited to it.
#
  1. SOB=$(git var GIT_AUTHOR_IDENT | sed -n ‘s/^\(.*>\).*$/Signed-off-by: \1/p’)
  2. grep -qs “^$SOB” “$1” || echo “$SOB” >> “$1”
  1. This example catches duplicate Signed-off-by lines.

test “” = “$(grep ‘^Signed-off-by: ‘ “$1” | sort | uniq -c | sed -e ‘/^[ ]*1[ ]/d’)” || { echo >&2 Duplicate Signed-off-by lines. exit 1
}
